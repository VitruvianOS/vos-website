## Download VitruvianOS

VitruvianOS is currently in active development. Pre-built nightly images are generated automatically from the latest source.

### Nightly Builds

Nightly images are the fastest way to follow current progress. They are built automatically on every significant commit and available via GitHub Actions.

[Download from GitHub Actions](https://github.com/VitruvianOS/Vitruvian/actions)

> **Note:** Nightly builds are for testing and development. Expect rough edges.

### Build from Source

Developers who want to contribute or explore the codebase can build the system from source:

[Build Instructions](https://wiki.v-os.dev/docs/getting-started/building/)

### Install

See the [How To Install](https://wiki.v-os.dev/docs/getting-started/how-to-install/) guide for writing images to USB drives and first boot instructions.

### System Requirements

- x86-64 processor (ARM and RISC-V support in progress)
- 2 GB RAM minimum (4 GB recommended)
- 10 GB disk space

### Getting Help

If you run into issues, open a ticket on [GitHub Issues](https://github.com/VitruvianOS/Vitruvian/issues) or ask in the [Telegram chat](https://t.me/vitruvian_official_chat).
